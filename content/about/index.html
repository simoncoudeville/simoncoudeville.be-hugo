Designer and teacher building internet software.

Hi, my name is Simon. I'm a designer into in responsive web-design, CSS, typography and animation. Currently I teach all of that stuff at MCT at Howest University. Some of my design work is on Dribbble. Some of my code experiments are on Codepen. You can send me an email or follow me on Twitter. If you're interested you can read more about me.

Technisch gezien is het misschien geen design system maar eerder een styleguide. While making this page I found it helped me take design decisions. It was a fun excersise to see if it was possible to make these typical guideline figures purely with CSS instead of images.
