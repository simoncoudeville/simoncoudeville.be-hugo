Type scale

Standard body text is 17px on small screens and 19px on larger screens. These pixel values are converted to percentages based on 16px, the default browser font size. Additional sizes are converted to rem. I don't use a modular scale. All additional sizes are handpicked and based on the baseline unit. So 4 lines of text set in one size equal 3 lines of text set in the next size.
